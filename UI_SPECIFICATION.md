## 2. 字体规范

### 2.1 字体家族

- **主字体**: `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif`
- **等宽字体**: `monospace`

### 2.2 字体大小

| 样式 | 大小 | 行高 | 用途 |
|-----|-----|------|------|
| 超大标题 | 48px | 1.2 | 登录页标题 |
| 大标题 | 32px | 1.3 | 页面标题 |
| 标题 | 24px | 1.3 | 卡片标题 |
| 正文 | 16px | 1.5 | 主要内容 |
| 小正文 | 14px | 1.5 | 辅助文字 |
| 小字 | 12px | 1.4 | 提示文字 |

## 3. 按钮规范

### 3.1 按钮类型

| 类型 | 样式 | 适用场景 |
|-----|------|---------|
| 主按钮 | 渐变背景、白色文字 | 主要操作 |
| 次按钮 | 透明背景、白色边框 | 次要操作 |
| 危险按钮 | 红色背景/边框 | 删除、退出等危险操作 |

### 3.2 按钮尺寸

| 尺寸 | 高度 | 圆角 | 字体大小 | 适用场景 |
|-----|-----|------|---------|---------|
| 大号 | 48px | 12px | 16px | 登录按钮等主要操作 |
| 中号 | 40px | 10px | 14px | 表单按钮 |
| 小号 | 32px | 8px | 13px | 工具栏按钮 |

### 3.3 按钮状态

| 状态 | 样式变化 |
|-----|---------|
| 正常 | 主色背景 |
| 悬停 | 主色深色、轻微上浮 |
| 点击 | 主色浅色、轻微下沉 |
| 禁用 | 透明度降低、光标禁用 |

## 4. 输入框规范

### 4.1 输入框样式

- **高度**: 48px
- **圆角**: 12px
- **边框**: 2px solid rgba(102, 126, 234, 0.65)
- **背景**: 深色渐变背景
- **文字颜色**: #ffffff
- **占位符颜色**: rgba(255, 255, 255, 0.7)

### 4.2 输入框状态

| 状态 | 样式变化 |
|-----|---------|
| 正常 | 默认边框 |
| 焦点 | 边框高亮、发光效果 |
| 错误 | 红色边框 |

| 间距名称 | 大小 | 用途 |
|---------|-----|------|
| 微型 | 4px | 元素内部 |

- **背景**: rgba(21, 21, 32, 0.95)
- **边框**: 1px solid rgba(255, 255, 255, 0.1)

## 8. 图标规范

- **图标大小**: 24px（标准）、36px（大号）
- **图标颜色**: rgba(255, 255, 255, 0.7)
- **悬停颜色**: #ffffff
- **禁用颜色**: rgba(255, 255, 255, 0.4)

## 9. 响应式断点

| 设备 | 断点 | 布局调整 |
|-----|-----|---------|
| 桌面 | > 1200px | 完整布局 |
| 平板 | 768px - 1200px | 中等布局 |
| 手机 | < 768px | 紧凑布局 |

## 10. 动画规范

### 10.1 过渡动画

| 元素 | 过渡时间 | 缓动函数 |
|-----|---------|---------|
| 按钮 | 200ms | ease-out |
| 卡片 | 300ms | ease-out |
| 模态框 | 300ms | ease-out |
| 页面切换 | 400ms | ease-out |

### 10.2 关键帧动画

- **浮动动画**: 上下轻微浮动，用于装饰元素
- **渐变动画**: 背景渐变流动效果
- **淡入动画**: 页面元素进入效果

## 11. 设计原则

1. **深色主题**: 整体采用深色背景，减少眼睛疲劳
2. **玻璃拟态**: 使用 backdrop-filter 实现毛玻璃效果
3. **发光效果**: 按钮、卡片边缘添加微妙的发光
4. **微动画**: 悬停、点击时有细腻的动画反馈
5. **一致性**: 所有交互元素保持统一的视觉语言
